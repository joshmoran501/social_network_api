## Installation:

`npm i`

## Usage:

Run `npm server` or `npm start` to spin up the server. Tp interacte with the API, use an application like [Insomnia](https://insomnia.rest/download) or [Postman](https://www.postman.com/downloads/) at localhost:3001.
